# Espresso ♨️

A NodeJS project setup tool

## Getting Started
Clone respository in your project folder

`python espresso.py`
  
### Architecture Support

- MVC
  - Dependencies Installed

    ```
    bcryptjs
    body-parser
    ejs
    express
    jsonwebtoken
    jwt-decode
    ```
   - Project Architecture
   
   ```
   
    ├── controllers
    ├── middleware
    ├── models
    ├── node_modules
    ├── public
    │   ├── css
    │   ├── script
    ├── routes
    ├── utils
    │   ├── db (If chosen)
    ├── views
    ├── package-lock.json
    ├── package.json
    ├── app.js
    └── .gitignore

    ```
